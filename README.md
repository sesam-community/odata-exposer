# sesam-odata-exposer
Exposes Sesam.io datasets as Odata

### env variable 
BASE_URL - Sesam instance (required)  
JWT_TOKEN - jwt token for this instance (required)  
LOG_LEVEL - default INFO  
PORT - default 5000
